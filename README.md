# InvenTree Custom Stock View Plugin

Плагін додає сторінку `/plugin/custom-stock/` з кастомним фільтром для Stock items.

**Installation (for admin)**:
- Admin → Plugins → Install Plugin
- Вставити URL репозиторію: `https://github.com/yourname/inventree-custom-stock-plugin.git`
- Після встановлення — Enable plugin
- Сторінка буде доступна: `/plugin/custom-stock/`
